#### for 循环

```php
<?php

    /* 语法格式:
        for(初始条件; 条件判断; 改变条件){
            // 循环体代码
        }

        执行步骤:
        1.初始条件
        2.条件判断   
        3.成立, 执行循环体
        4.改变条件, 执行第2步

        第2步 条件不成立,结束循环

    */

    for($i = 1; $i <= 10; $i++){

        echo $i,'<br>';  // 循环输出 1 ~ 10

    }


?>
```

       初始化条件

       }

?>
