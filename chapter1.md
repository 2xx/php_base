# Hello PHP

```php
<?php

    // 输出显示 hello php!
    echo 'hello php!';

?>
```

#### 

#### **定界符：**

> **&lt;?php**     表示代码开始
>
> ... ...
>
> **?&gt;**            表示代码结束

#### 

#### 注释：

作用：1）对旁边代码进行解释说明    2）使原来的代码不被执行

> **//** 单行注释
>
> **\#**_**  **_脚本注释, 也是单行注释
>
> **/\***   ... ...   **\*/**      多行注释

####   

#### 输出语句：

> **echo**  后面的内容将会被输出显示
