#### 正则表达式中的 子模式

```php
<?php

    // 正则表达式
    $ptn = '/<img src=.*?\/>/';    // 匹配整个标签
    $ptn = '/<img src=(.*?)\/>/';  // 匹配整个标签, 小括号中的内容会生成一个子数组, 称为子模式

    // 目标字符串
    $str = "<img src='./a/b.jpg' />";

    // 执行匹配函数
    preg_match_all($ptn, $str, $arr);

?>
```

#### 子模式的一个用法  \1

```php
<?php

   /*
       需求:  把 2017-03-20  转换成 03/20/2017
   */

    // 正则表达式
    $ptn = '/(\d{4})-(\d{2})-(\d{2})/';

    // 目标字符串
    $str = '2017-03-20';

    // 执行匹配替换
    echo preg_replace($ptn, '\\2/\\3/\\1', $str);

    /*
        \1  代表正则表达式中 第一个小括号  匹配到的内容
        \2  代表正则表达式中 第二个小括号  匹配到的内容
        \3  代表正则表达式中 第三个小括号  匹配到的内容
    */
    
?>
```
