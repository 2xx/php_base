#### 变量函数

把**函数的名称**放在一个变量中，之后可以通过变量调用这个函数。

```php
<?php

    $a = 'test';   // 1.把函数名称存放在变量中
    $a();          // 2.通过变量调用函数

    function test()
    {
        echo '我是test函数!';
    }

    // 变量函数的方式, 让代码书写更加灵活，一期项目中会用到
?>
```

#### 匿名函数

把**函数的定义**放在一个变量中，之后可以通过变量调用这个函数。

```php
<?php

    // 把函数的定义赋值给一个变量, 这个函数可以没有名字
    $a = function(){
        echo '我是一个匿名函数';
    };   // 注意最后的分号不能少
    
    
?>
```
