### 函数 function

**1）一段代码**

**2）多次调用执行。 函数定义后不会执行，调用才执行。**

```
  语法格式:

        1.定义函数

        function 函数名(参数列表)
        {
            ... ... 函数体代码
        }


        2.调用函数

        函数名(参数);  // 执行函数中的具体代码
```

```php
<?php

    test();  // 调用执行函数   注意:函数会被提前加载到内存, 所以调用函数可以写在函数定义之前

    function test()
    {
        echo '一个测试函数';
    }


?>
```
