### 变量的销毁

所有变量都会在 php文件代码结束时，**自动销毁**。

### 为了及时释放内存

##### 1.变量不用时，可以被提前销毁

```php
<?php

    $a = 7;

    unset( $a );

    // 下面有1000行代码，但不会用到$a

?>
```

##### 2.也可以给变量赋值为 null

```php
<?php

    $a = 7;

    $a = null;

    // 下面有1000行代码，但不会用到$a

?>
```

**扩展知识：**

```
   如果有10000个人同时执行php文件，那么在内存中会有10000个 $a 变量

   所以及时释放内存是非常有必要的.
```
