#### 隐式类型转换

根据运算符的需要，一个类型的数据可能会被自动的转化为另一种类型。

```php
<?php

    echo '2C1' + '3b8' + '1.5a'; // 转化为 2+3+1.5  输出显示 6.5

    echo true + false;  // 转化为 1+0  输出显示 1

    echo true;  // 输出显示   1
    echo false; // 转化为空，什么也不显示


    /*
        一些数据会被转化为 false

        ''    '0'
        0
        0.0
        false

        []
        null
        未定义的变量

    */ 


?>
```
