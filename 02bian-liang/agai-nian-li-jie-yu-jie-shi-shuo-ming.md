# 什么是变量？

```php
<?php
        $a = 'i  love  you';   // 1.存储内容

        echo  $a,'<br>';  // 使用变量中的内容
        echo  $a,'<br>';       // 2.多次使用

        $a = 'i  miss  you';   // 3.新的内容会覆盖旧的内容
        echo  $a;
?>
```

```php
<?php
        $a = 7;     // 把7存入 $a
        $b = $a;    // 把$a的内容存入 $b    ($a中的内容还在)

        echo $a;    // 输出7   这也说明变量的内容可以多次使用
        echo $b;    // 输出7
?>
```
