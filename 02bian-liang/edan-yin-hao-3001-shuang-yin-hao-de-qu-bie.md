## 单引号和双引号的区别

#### 1.单引号不能解析变量

```php
<?php

    $name = '景水';
    echo '我们的讲师叫$name';  // 输出显示 我们的讲师叫$name
                              // 因为变量出现在单引号定义的字符串中，不能解析

?>
```

#### 2.双引号可以解析变量

```php
<?php

    $name = '景水';
    echo "我们的讲师叫$name"; // 输出显示 我们的讲师叫景水
                             // 变量出现在双引号定义的字符串中，可以被解析

?>
```
