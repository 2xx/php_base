### 一. 类型约束

​ 类型约束, 是调用成员方法时, 对**参数类型**加以限制.

* 如果约束的类型为**四种标准类型**, 当调用时参数的类型不对, 会尝试类型转换

```php
	class A
    {
     	static function xxoo(int $a)   // 约束 参数应该是 整数型
        {
             echo '<pre>';
             var_dump( $a );    
        } 
    }
	A::xxoo('2b');    // 2 因为类型约束   函数中会尝试进行类型转换
    A::xxoo(true);    // 1
    A::xxoo(2.34);    // 2
```
