#### 赋值运算符

```php
<?php

    $a = 7;   // 把7赋值给$a, 数据是从右向左的
    
    $a += 7;  // 相当于 $a=$a+7
    
    $a -= 7;  // 相当于 $a=$a-7
    
    $a *= 7;  // 相当于 $a=$a*7
    
    $a /= 7;  // 相当于 $a=$a/7
    
    $a %= 7;  // 相当于 $a=$a%7


    $a = 7;
    $a .= 2;
    echo $a; // 72  .是字符串连接符, 所以$a就变成了字符串的72了

?>
```
