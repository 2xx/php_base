#### 逻辑运算符

逻辑运算的结果是  **布尔值**

**逻辑与    &&**

```php
<?php

    //  与是并且的意思, 运算符两边同时为真, 结果才为真
    var_dump( 5 > 1  &&  2 < 7 ); // true     真&&真  真
    var_dump( 5 < 1  &&  2 < 7 ); // false    假&&真  假
    var_dump( 5 > 1  &&  2 > 7 ); // false    真&&假  假
    var_dump( 5 < 1  &&  2 > 7 ); // false    假&&假  假
    
?>
```
