#### 与短路

```php
<?php

    $a = 1;
    $b = (5 < 3) && ($a = 3);  
    var_dump( $a, $b ); // 1  false

    /* 分析:
    
        5 < 3 为假   
        整个与运算的结果就为假, 所以 ($a = 3) 就不需要执行了 
    
    */

?>
```

#### 或短路

```php
<?php

    $a = 1;
    $b = (5 > 3) || ($a = 3);
    var_dump( $a, $b ); // 1  true
    
    /* 分析:
       
       5 > 3 为真
       整个或运算的结果就为真, 所以 ($a = 3) 就不需要执行了
    
    
    */

?>
```
