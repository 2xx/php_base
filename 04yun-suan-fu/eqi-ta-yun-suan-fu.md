#### 三元运算符

```php
<?php

    /*
        + 表示两个数相加, 称为 二元运算符
        三元, 则是指有三个操作数.

        表达式1 ?  表达式2 : 表达式3

        表达式1为真  取表达式2的值
        表达式1为假  取表达式3的值
    */

    // 5==2为假 取第3个值
    echo 5 == 2 ? '相等' : '不相等';

?>
```

#### 错误抑制符

```php
<?php
    
    echo @$a;  // 未定义变量, 应该报错, 但加上@就不会把错误显示在页面了

?>
```
