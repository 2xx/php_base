#### 自增运算

```php
<?php

    $a = 5;
    $a++;    // 相当于 $a=$a+1
    echo $a; // 6  


    $a = 5;
    ++$a;    // 相当于 $a=$a+1
    echo $a; // 6

?>
```

#### $a++ 与  ++$a 的区别

```php
<?php

    $a = 5;
    echo $a++; // 5   先使用$a原有的值输出，再让$a加1
    echo $a;   // 6
    
    
    $a = 5;
    echo ++$a; // 6   先让$a加1，再使用$a的值输出
    echo $a;   // 6

?>
```
